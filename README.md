# Clinic Queue System - Naryn

A medical queue management system for patients and doctors, built with Node.js, Express, and vanilla JS.

## Features
- Patient registration with queue numbers
- Live status updates: Waiting → In Progress → Done
- Doctor dashboard with Call Next button
- Multi-day scheduling
- Language switch (English, Russian, Kyrgyz)

## Run Locally
1. Clone the repo: `git clone <repo_url>`
2. Install dependencies: `npm install`
3. Run server: `node server.js`
4. Open browser: `http://localhost:3000`
